# uninvited archive

a partial recovery of a private minecraft server's discord.
the server was deleted. someone made a copy before it was gone.

---

this repository contains preserved logs, member records, and saved data
from a server that no longer exists. the original owner has been unreachable
since the deletion date.

files were recovered from a local cache. some content could not be restored.

---

### contents

- `archive.html` — recovered channel logs and pinned content
- `members.html` — member list at time of deletion
- `puzzle.html` — an encoded document found in the server
- `locked.html` — access restricted

---

### notes
