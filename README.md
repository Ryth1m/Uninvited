# uninvited archive

a partial recovery of a private minecraft server discord and world.
the server was called **reedworld**. it ran on java 1.8.9.
it was deleted after 847 days of continuous uptime.

this archive was made without the permission of root__.

---

### what is this

someone kept a minecraft server running for 847 days waiting for a person who never came back.

a player named gravelpit found the server by accident.
he kept returning.
root__ kept trying to make him leave.

this archive was put together from recovered cache files after the server went offline.
it is not complete. some files were corrupted. some were deleted before they could be saved.
